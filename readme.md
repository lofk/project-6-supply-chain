## Supply Chain Project

### Requirements
- NodeJS v8.11.4
- Truffle v4.1.14 (core: 4.1.14)
- Solidity v0.4.24 (solc-js)

### Install Application
```
npm install

# compile the contracts
truffle compile
```

### Run tests
```
truffle test
```

### Development
```
npm run dev
```
Make sure you have 4 different wallets for each stakeholder, to interact with the smart contract.
